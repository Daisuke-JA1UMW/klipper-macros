# klipper-macros
Extension for Klipper 3D Printer Controller

## temp_margin.cfg
目的 purpose:

・Klipperの温度待ちコマンド M109 M190 は設定温度ピッタリに一致するまで進行しないため、プリント開始に時間がかかる事がある。

機能:

・温度待ちコマンド M109, M190 をマージン付きの判定処理にして、待ち時間を早めます。

・Mainsailのコンソールに温度設定方向を絵文字で表示します。
